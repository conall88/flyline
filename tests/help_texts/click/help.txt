Usage: cli [OPTIONS] COMMAND [ARGS]...

  A simple CLI built with Click.

Options:
  --name TEXT      The person to greet.
  --count INTEGER  Number of greetings.  [default: 1]
  -v, --verbose    Enable verbose output.
  --help           Show this message and exit.

Commands:
  hello    Greet someone.
  goodbye  Say goodbye.
